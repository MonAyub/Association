# Images Asset Directory

Place your custom website images in this folder. Ensure the filenames match the references below or modify `index.html` to target your files.

## Suggested Filenames & Recommended Dimensions:

1. **president.jpg** (1:1 aspect ratio, e.g. 400x400px)
   - President's portrait photo for the Home and Committee sections.

2. **gs.jpg** (1:1 aspect ratio, e.g. 400x400px)
   - General Secretary's portrait photo for the Home and Committee sections.

3. **vp.jpg**, **js.jpg**, **os.jpg**, **treasurer.jpg**, **cultural.jpg** (1:1 aspect ratio, e.g. 400x400px)
   - Portrait photos for other executive committee members.

4. **event_freshers.jpg**, **event_picnic.jpg**, **event_sports.jpg** (16:9 aspect ratio, e.g. 800x450px)
   - Cover photos for the events section cards.

5. **gallery_1.jpg** to **gallery_6.jpg** (4:3 aspect ratio, e.g. 800x600px)
   - Photos for the responsive Photo Gallery grid.

## Fallback Design

If no images are placed in this folder, the website's CSS and JS are designed to fall back gracefully to a premium dark-glass UI, presenting vector icon placeholders and ambient gradients automatically. This ensures the website remains visually stunning out-of-the-box!
